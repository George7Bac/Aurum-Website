# Aurum-Website
Aurum adalah website aplikasi pemutar musik (music streaming platform) yang dirancang dengan tampilan modern dan premium. Website ini memungkinkan pengguna untuk mendaftar akun, masuk ke sistem, mencari lagu, mengelola playlist, menyukai lagu, serta menikmati pengalaman mendengarkan musik melalui fitur pemutar musik yang lengkap.
